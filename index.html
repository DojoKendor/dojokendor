Conexión lista con tus llaves. Prueba crear cuenta o entrar desde la consola:

  const s = supabase.createClient(window.DOJO_SUPABASE_URL, window.DOJO_SUPABASE_ANON);
  await s.auth.signUp({ email: "tu@correo.com", password: "clave123" });
  await s.auth.signInWithPassword({ email: "tu@correo.com", password: "clave123" });
